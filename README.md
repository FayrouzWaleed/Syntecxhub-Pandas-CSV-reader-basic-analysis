# Pandas CSV Reader & Basic Analysis

## Description
This project demonstrates how to use **Python pandas** to read a CSV file, inspect the data, perform basic statistical analysis, filter and select data, and save the results to a new file.

The project focuses on practicing core pandas operations commonly used in data analysis.

---

## Dataset
- The dataset used in this project is **people-100.csv**.
- It contains information about 100 individuals, including:
  - First Name
  - Last Name
  - Gender
  - Email
  - Phone Number
  - Date of Birth
  - Job Title

---

## Tasks Performed
- Read the CSV file into a pandas DataFrame
- Inspect the dataset using:
  - `head()`
  - `tail()`
  - `dtypes`
- Compute summary statistics such as:
  - mean
  - median
  - minimum
  - maximum
  - count
- Filter rows based on specific conditions
- Select and slice specific columns
- Save the filtered dataset to a new CSV file
